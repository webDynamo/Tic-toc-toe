## 🎮 Tic Tac Toe Game
A simple web-based Tic Tac Toe game built using HTML, CSS, and JavaScript. Play against a friend locally in this classic two-player game!

## Features
🧠 Two-player mode

🎨 Clean and responsive UI

🔁 Reset game button

✅ Winner detection and draw condition
